An NLP Model to Predict a Subreddit Post as Car or Bike

Web scraping is an integral process that helps in quick and effective extraction of news data from different sources. Reddit is a social news platform that allows users to discuss and vote on content that other users have submitted. NLP or natural language processing is important because it helps resolve ambiguity in language and adds useful numeric structure to the data. In this project, data is scraped from reddit with pushshift API, then data is cleaned and NLP is done. In the modelling phase Random Forest and Linear Regression models are used. It will help users to identify which model is accurate based on test scores and train scores.

Ocular Disease Recognition Using Neural Network

Our team set out to build a convolutional neural network that would classify healthy and unhealthy eyes based on their fundus images. We initially set out to create a multi-class model, but soon realized we needed to switch to separate binary classification models for each disease because some of the images were of eyes that had multiple diseases. After switching to binary classification models, our accuracy increased to expected levels in the 85-95% range over the course of training. My contribution towards this project includes exploratory data analysis (EDA) in which various relation and behaviours of the features were analysed and important predictive features were identified. Then a CNN model is built and trained, which extracts features from the input images and classifies them accordingly.
